# easyweather
个人练手小demo，主要功能如下：
* 定位手机所在位置，并显示当地城市天气
* 添加多个城市，可在主ViewPager 左右滑动查看天气
* 能显示未来10天的温度天气信息
* 能显示未来24小时的天气温度信息
* 调用的api使用期已过期，现在为免费版，部分信息可能无法正常显示
